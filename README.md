This full stack project was made using the following technologies:

- Front End:
  - Axios
  - Bootstrap
  - Emoji Picker
  - Font Awesome
  - React
  - Socket.IO
  - TinyMCE

- Back End:
  - Bcrypt
  - Cheerio
  - Cookie parser
  - Cors
  - Express
  - JSON Web Token
  - Mongoose
  - MongoDB Atlas 
  - Node cron
  - Socket.IO
